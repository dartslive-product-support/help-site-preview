    dl2_001_Unable to start / Screen does not display anything

    1. Are the dart machine's lights on?

    Check the dart machine power cable and its power switch.
    If the cable is connected properly and the power switch is on, go to Step2.

    2. Is the PC power on?

    Make sure the PC power cable is connected properly and then press the power button.
    If the PC lights are on, go to Step3.

    Picture: Locations of the power button

    3. Has the PC started yet?

    If it beeps when switched on, the system is working fine.
    If it keeps beeping, there might be a defect in the PC.
    Please contact our sales representative.

    4. Is the monitor power on?

    1)Make sure the power cable is properly connected and the power button is on.
    If there is no problem with the power supply, go to 2).

    Picture: Power button at the right bottom of the monitor

    2) Check the VGA cable to ensure that
    it is properly connected (not loose) and also check to see if the line has any damage.
    Connect the PC to other dart board if possible.

    The other dart board starts showing the same symptom → the problem is in the PC.
    If the symptom doesn't appear on the other dart board → the problem is in the cables.

    If none of the above solves the problem, there might be a defect in the PC or the power supply.
    Please contact our sales representative.

    Reference : Page 50 of the Support Manual「起動しない モニターが映らない」
